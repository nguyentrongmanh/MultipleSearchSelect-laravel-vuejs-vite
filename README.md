## Required enviroiment version

$ php >= 8.0.x
$ node >= 16.x

## Installation

``` bash
# install app's dependencies
$ composer install
$ npm install
# generate laravel APP_KEY
$ php artisan key:generate
# run database migration
$ php artisan migrate
```
## Usage

``` bash
# start local server
$ php artisan serve
# start vite server
$ npm run dev
```
